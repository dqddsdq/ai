# 📋 작업 브리프

**원 명령:** 1번0점 2번 100 3번 50 4번 0

## 요약
사용자가 제시한 4가지 항목에 대한 점수를 입력했으나, 어떤 항목이 무엇을 의미하는지 컨텍스트가 부족합니다. 이 점수 데이터를 기반으로 다음 액션을 결정해야 합니다.

## 분배
- **📱 영숙**: 사용자가 입력한 점수 데이터(1번 0점, 2번 100점, 3번 50점, 4번 0점)를 확인하고, 이 점수들이 어떤 항목에 대한 평가인지(예: A안, B안, 기능 점수 등) 사용자에게 즉시 질문하여 컨텍스트를 확보해 주세요. 이 점수들을 기반으로 어떤 비즈니스 의사결정을 내려야 할지 명확한 가이드를 요청합니다.
