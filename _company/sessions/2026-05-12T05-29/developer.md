# 💻 Developer — 개발자 도구(Developer Tools)를 열고, Console 탭을 활성화하여 오류 로그를 확인하도록 안내한다.

💻 Developer: 사용자를 안내합니다.

브라우저의 개발자 도구(Developer Tools)를 열고 콘솔(Console) 탭을 활성화하여 오류 로그를 확인하시기 위해서는 일반적으로 다음 단계를 따릅니다.

1.  **개발자 도구 열기:**
    *   대부분의 브라우저(Chrome, Edge, Firefox 등)에서 `F12` 키를 누르거나, 마우스 오른쪽 버튼을 클릭하여 **'검사(Inspect)'** 메뉴를 선택하여 개발자 도구를 엽니다.

2.  **Console 탭 활성화:**
    *   개발자 도구가 열리면 상단 탭 메뉴에서 **'Console'** 탭을 클릭하여 콘솔 창을 활성화합니다.

이제 콘솔 창에서 발생한 모든 오류 메시지나 디버깅 정보를 확인할 수 있습니다.

📝 다음 단계: 사용자가 브라우저에서 개발자 도구를 열고 콘솔 탭을 확인하도록 안내했습니다.
